## 配置详解

|  参数   | 说明  | 必传  |默认值  |
|  ----  | ----  |----  |----  |
| source  | 类型 |是 |MQTT  |
| sourceName| 数据源名称 | 是|无  |
| sourceAttributes| 额外参数 | 否|无  |

```markdown
smqtt:
  sources: # 配置数据源sources
    - source: MQTT # mqtt配置
      sourceName: mqtt
      sourceAttributes:
        host: 123.57.69.210
        port: 1883
        clientId: testClient # 客户端标志
        userName: smqtt      # 鉴权帐号
        passWord: smqtt		 # 鉴权密码
```

> sourceAttributes必须设置host、port、clientId，userName和passWord选配
